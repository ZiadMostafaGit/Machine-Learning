## Week 3: Hands-On Linear Regression Project

In the third week of our machine learning journey, we will start to get hands-on and do a project on linear regression. Linear regression is a linear approach for modeling the relationship between a scalar response and one or more explanatory variables (also known as dependent and independent variables). The case of one explanatory variable is called simple linear regression; for more than one, the process is called multiple linear regression.

We will start by collecting data sets from sources such as Kaggle, Google Dataset Search, and Tableau Public Data Sets. These sources provide a wide range of data sets that we can use to practice and apply our knowledge of linear regression.

Once we have collected our data sets, we will begin to familiarize ourselves with linear regression models. We can use resources such as the scikit-learn documentation to learn about the implementation of linear regression in Python. Additionally, we can refer to resources such as Scribbr for an easy introduction and examples of simple linear regression.

By the end of this week, we should have a good understanding of linear regression and be able to apply it to our collected data sets.
